# 🧬 MGG Tools 🛠️

**MGG Tools** es tu centro de recursos no oficial y definitivo para el popular juego de estrategia **Mutants: Genetic Gladiators** de Celsius Online.  
Calcula, compara y domina la arena con nuestras herramientas especializadas diseñadas para maximizar tu estrategia de juego.

---

## ✨ Características Principales

MGG Tools proporciona un conjunto de herramientas esenciales para la comunidad de Mutants: Genetic Gladiators:

* **Stats Simulator** — Calcula las estadísticas exactas de cualquier mutante a cualquier nivel deseado.
* **Mutants List** — Explora y filtra la lista completa de todos los mutantes disponibles en el juego.
* **Evo Calculator** — Determina el costo preciso de Oro y Créditos de Evolución necesarios para subir de nivel a tus mutantes.
* **Top Mutants** — Consulta un ranking de los 10 mutantes más fuertes por cada gen, basados en sus estadísticas base.
* **Damage Simulator** — Calcula el daño exacto que le harías a un mutante considerando sus stats y las del rival.
* **Mutant Builder** — Constructor completo de mutantes: estadísticas, estrellas, genes, habilidades y edición manual.
* **Compare Mutants** — Compara las estadísticas de hasta 3 mutantes lado a lado en cualquier nivel.
* **Breeding Simulator** — Descubre qué mutantes pueden salir al cruzar dos padres: normales, bingos, legendarios y secretos.
* **How to get Mutants** — Guía completa de obtención: encuentra el origen de cada mutante (tienda, raid, reactor, bingo, exchange, eventos y más) con filtros, búsqueda y paginación.
* **Rebalance** — Revisa los cambios recientes en las estadísticas o habilidades de los mutantes.
* **Changelog** — Mantente al día con todas las versiones y novedades del proyecto.

## 💻 Desarrollo

Este proyecto está construido con tecnologías web front-end puras para ofrecer una experiencia rápida y accesible sin necesidad de backend:

* **Lenguajes:** HTML5, CSS3, JavaScript (ES6+)
* **Tipografía:** Rajdhani, Barlow Condensed, Space Mono (Google Fonts)
* **Estilo:** Sistema de variables CSS con soporte para tema claro/oscuro persistente
* **Internacionalización:** Sistema i18n propio con soporte ES / EN
* **Datos:** CSV externo vía fetch + variables JS integradas (sin JSON externos)
* **Diseño:** Interfaz oscura tipo dashboard con sidebar colapsable
